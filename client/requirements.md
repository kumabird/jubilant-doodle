## Packages
matter-js | Physics engine for the eraser drop mechanics
@types/matter-js | TypeScript definitions for matter-js
framer-motion | Playful, bouncy page transitions and micro-interactions
lucide-react | Icons for menus and UI
zustand | Simple global state management for game setup

## Notes
- Using `matter-js` for top-down 2D physics. Gravity will be set to 0.
- Canvas-based custom rendering layered over matter-js.
- "Eraser Drop" is a turn-based slingshot game.
